Project List

This is a list of topics proposed by current graduate students interested in mentoring this semester.

Applied Math

  • (Opinion Dynamics) This project will be on operation dynamics which uses probability theory, linear algebra, and graph theory to model how opinion flows in a population.

Algebra

  • (Commutative Algebra) This project will be about mixed characteristic commutative algebra.
  • (Galois Theory) This project will work with problems in Galois Theory, which is a branch of math that builds a connection between group and field theory.
  • (Commutative Algebra, Homological Algebra, Algebraic Geometry) This project will work on problmes in any of the topics commutative algebra, homological algebra, or algebraic geometry.
